# Інформаційна система онлайн-запису до лікаря на основі попереднього діагнозу

Демонстраційна програмна реалізація до бакалаврської кваліфікаційної роботи спеціальності 126 «Інформаційні системи та технології».

## Функції
- реєстрація та авторизація користувачів;
- рольова модель: пацієнт, лікар, адміністратор;
- введення симптомів або попереднього діагнозу;
- правиловий алгоритм рекомендації медичної спеціальності;
- вибір лікаря відповідної спеціальності;
- вибір доступного часового слоту;
- створення запису та автоматичне блокування зайнятого слоту;
- перегляд створених записів;
- адміністративне керування довідниками.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python seed_data.py
python app.py
```

Після запуску відкрити у браузері: http://127.0.0.1:5000

## Тестові користувачі
- Пацієнт: `patient@example.com` / `patient123`
- Лікар: `doctor@example.com` / `doctor123`
- Адміністратор: `admin@example.com` / `admin123`

## Важливе обмеження
Система не встановлює медичний діагноз. Рекомендація спеціальності має інформаційний характер і не замінює консультацію лікаря.
